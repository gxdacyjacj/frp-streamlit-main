üì¶ REQUIRED FILES

1. Model file (.pkl) - Required

================================================================================
üìÑ 1. MODEL FILE REQUIREMENTS
================================================================================

File Extensions:
‚úÖ .pkl (recommended)
‚úÖ .pickle

Content Requirements:
‚úÖ Must be Python pickle serialized object
‚úÖ Model object must have predict() method
‚úÖ Supports sklearn, XGBoost, LightGBM models

Supported Model Types:
‚úÖ scikit-learn: RandomForest, DecisionTree, ExtraTrees, etc.
‚úÖ XGBoost: XGBRegressor, XGBClassifier
‚úÖ LightGBM: LGBMRegressor, LGBMClassifier
‚úÖ Custom models with predict() method

Example:
```python

from sklearn.ensemble import RandomForestRegressor

model = RandomForestRegressor(n_estimators=100)
model.fit(X_train, y_train)

with open('my_model.pkl', 'wb') as f:
    pickle.dump(model, f)
```

================================================================================
üîß 2. PREPROCESSOR FILE REQUIREMENTS  
================================================================================

File Extensions:
‚úÖ .pkl (recommended)
‚úÖ .pickle

Content Requirements:
‚úÖ Must be Python pickle serialized preprocessor object
‚úÖ Must have transform() method
‚úÖ Supports sklearn preprocessors

Supported Types:
‚úÖ StandardScaler, MinMaxScaler, RobustScaler
‚úÖ Pipeline, ColumnTransformer objects
‚úÖ Custom preprocessors with transform() method

Recommended for:
‚ö†Ô∏è Neural Network (MLP) - Required
‚ö†Ô∏è Support Vector Machine - Strongly recommended  
‚ö†Ô∏è Linear Models - Recommended
‚úÖ Tree models - Optional

================================================================================
üè∑Ô∏è 3. FEATURE CONFIGURATION
================================================================================

Format:
- One feature name per line
- Must match training features exactly
- Case sensitive
- No empty lines

Example:
```
pH
Fiber_content_wt
Diameter_mm
Temp_C
Duration_hours
```

================================================================================
üöÄ 5. UPLOAD STEPS
================================================================================

1. Prepare .pkl model file
2. Prepare .pkl preprocessor file (if needed)  
3. Go to "Model Configuration" tab
4. Check "Use Custom Pre-trained Model"
5. Upload files and configure features
6. Click "Validate and Save Custom Model"

================================================================================
‚ö†Ô∏è 6. COMMON ISSUES
================================================================================

Unsupported Formats:
‚ùå .h5, .joblib, .json, .xml files
‚ùå TensorFlow, PyTorch native formats
‚úÖ Only Python pickle (.pkl) supported

Feature Mismatch:
‚ùå Different feature names ‚Üí Prediction failure
‚ùå Different feature count ‚Üí Error
‚úÖ Ensure exact feature name and order match

Preprocessor Issues:
‚ùå Mismatched preprocessor ‚Üí Poor predictions
‚úÖ Use same preprocessor as training
‚úÖ Neural networks and SVM strongly need preprocessor
